# Task: iOS Custom Instrumentation and Context

## Description
Implement comprehensive custom instrumentation throughout the iOS app to capture detailed context about user interactions, Live Activity lifecycle events, API calls, and performance metrics. This provides rich debugging information and performance insights specific to TrackRat's functionality.

## Acceptance Criteria
- [ ] Custom breadcrumbs added for Live Activity lifecycle events (start, update, end)
- [ ] APIService.swift instrumented with network spans and detailed error context
- [ ] User context configured with selected stations and environment information
- [ ] Navigation flow tracking implemented across key ViewControllers
- [ ] User interaction tracking for critical app features
- [ ] Performance monitoring for screen load times
- [ ] Background task operations instrumented
- [ ] Train status update operations tracked
- [ ] Location permission and usage tracking

## Technical Details
- Add Sentry.addBreadcrumb() calls throughout Live Activity lifecycle
- Instrument network requests in APIService with custom spans
- Set user context with station selections, preferences, and environment
- Add navigation breadcrumbs in key ViewControllers
- Instrument critical user flows (station selection, Live Activity creation)
- Monitor performance of background refresh operations
- Track API response times and failure rates
- Add custom tags for debugging (iOS version, device type, etc.)
