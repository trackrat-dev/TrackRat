# Universal Links for TrackRat

Universal Links deployment has been moved to keep the root directory clean.

## 📁 Location

All Universal Links files are now in:
```
universal-links-deployment/
```

## 🚀 Quick Start

```bash
cd universal-links-deployment
export PROJECT_ID="your-trackrat-project-id"
./deploy-universal-links.sh
```

## 🎯 What It Does

Enables sharing train URLs like:
```
https://trackrat.net/train/NJT-3849?from=NY&to=TR
```

That open directly in the TrackRat iOS app when tapped!
